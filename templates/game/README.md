# VengaiCode Web Game Template

This template is a lightweight web game shell built with React Three Fiber and Three.js.

## How to use

1. Install dependencies:

```bash
pnpm install
```

2. Start the game locally:

```bash
pnpm dev
```

3. Build for production:

```bash
pnpm build
```

## Notes

- This template is intended as a game target for generated projects.
- It is not Unreal Engine, but it is a modern web 3D engine path equivalent for browser-based games.
- For native Unreal support later, add a separate `templates/unreal` folder with Unreal project scaffolding.
