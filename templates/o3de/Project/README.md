# O3DE Project Skeleton

This folder contains the minimal skeleton for an O3DE game project.

- `Assets/` should contain scene assets, materials, and prefabs.
- `Scripts/` should contain game logic modules or Python build scripts.
- `ProjectName.project` and `ProjectName.workspace` are the root O3DE project files.

To make this real, replace these placeholders with a valid O3DE project export.
